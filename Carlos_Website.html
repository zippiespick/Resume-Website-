Unbound

Unbound is an alpha version of a game concept created by Eduardo Dotel(a friend of mine) and this website creator. The game will take place in 18th century Cuba and would see a soldier fighting a werewolf clan, coinciding with the invasion of the country by the English in 1741. The game is conceptualized to follow the trends created by game giants such as Doom( 1993 and 2016), Wolfenstein, Blood, etc. To accomplish this vision, we used free assets to create the game's code from scratch.
